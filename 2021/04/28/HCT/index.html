Modeling hierarchical category transition for next POI recommendation with uncertain check-ins

一：写作动机：

1：被命名为集体POI的地方（例如购物中心）留下一些不确定的签入。 它表明，我们不能总是访问用户在集体POI上的精确签入，因此现有的方法无法很好地工作。

2：不确定的签入加剧了冷启动问题，因为在用户的历史签入中无法观察到集体POI中的单个POI。

3：类别层次结构(CHS)在解决冷启动问题方面是有效的，类别层次将有助于更好地捕捉POI（类别)和类别之间的关系，从低(叶)到高(根）级别，并说明低层的类别涵盖比上层更具体的概念。

二：模型和方法：

1.本文通过RLMS（Representation learning model based approaches.）而不是DLMS（Deep learning model based approaches.）或LFMS（Latent factor model based approaches.）对层次类别转换进行建模，因为RLMS在排序精度和计算成本之间实现了相对的平衡。

2.在训练模块中，扩展了the Skip-gram以建模POI（类别）和类别之间的层次依赖关系，以及层次类别转换，从而学习了高质量下一个POI推荐更好的POI和类别表示。the Skip-gram (Word2Vc)模型起源于自然语言处理(NLP)域，这是一种通过捕获大量精确的语法和语义单词关系来学习高质量单词分布式向量表示的有效方法。

3：用soft-max functions:预测Li在第一层(Clli)与其相关类别的概率依赖关系，以及在第j层(Clji)到第(J1)层(Clji1)的类别的概率依赖关系。

4：进行top-k个poi推荐，top-k个类别推荐，个人top-k推荐（根据起初top-k的序列进行地理因素的限制：地理中心的距离按升序排列）

三：总结：

通过利用：（1）CH不同层次的类别转换和（2）CH中POI（类别)和类别之间的层次依赖，提出的HCT能够捕获用户的偏好转换模式，并很好地解决集体POI不确定签到（集体POI中的单个POI都是冷启动POI）引起的冷启动问题。

文章的创新点：

1.提出了一个新的研究问题，即向集体POI签到不确定的用户预测下一个精确的单个POI。据我们所知，这是第一个研究这一现实问题的工作。

2.我们开发了一个新的层次类别转换(HCT)框架，它利用不同层CH上的各种类别转换，以帮助捕捉不同粒度下的用户偏好转换。此外，采用POI（类别）和类别之间的层次依赖关系来解决冷启动问题，即推荐集体POI中的单个POI。
